Etherframe project page

Etherframe is a simple ethernet frame receiver with bare protocol decoding and frame statistics. The intended use is to try capture of signal quality. Multiple factors generally concur to signal quality.

Etherframe currently builds and run on Fedora.

Starting from version 0.1.4, Etherframe includes a prototype support for COP and ROP system control variables introduced in the Preventive Maintenance Model, so that we can produce surveyed data and actually measure experiments. In the network case, the COP is computed directly from the surveyed data given on input by the network interface card.

Theory of Operation

Terminology

Definition of accuracy

The degree of conformity with a standard, or a value accepted as correct or true. For measured operational data the accuracy is normally expressed in terms of a quantity of a stated resolution within which there is a defined confidence of the true resolution falling.

Definition of precision

A measure of the tendency of a set of random numbers to cluster about a number determined by the set. Usual measure: Standard deviation with respect to the average or its reciprocal.

Definition of resolution

The smallest difference between two adjacent values which can be represented in a measuring system. The number of decimal points or the scale of units to which a measured or computed data item can be recorded, displayed or transferred.

Quality of data

The terms 'precision' and 'resolution' are often interchangeable in general use. Here they stand for a measure of the data field capacities that are available within a specific system (example: 54º 33' 15" is expressed to a resolution of one second). Any process that manipulates data subsequent to the original measurement or definition cannot increase the precision to which the data were originally measured or defined, regardless of the resolution available within the system itself.
